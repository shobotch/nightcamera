NightCamera
===========

名前の由来に特に理由はありません。

使い方
----

無音で撮影して、`mnt/SDCARD/アプリ名/yyyyMMddkkmmss.jpg`に保存されます。

起動すると真っ白な画面の真ん中にボタンがあり、そこを押すとカメラのプレビュー画面が起動します。

どこでもいいので画面をタッチするとオートフォーカスが働いて、自動的に撮影されます。

注意
----

まだバグが数多く残されていますので実用には絶えませんが、無音カメラの仕組みを知るには最適化と思われます。
